# **App Name**: InstaPay QR

## Core Features:

- Amount Input: Allows Aditya to input the payment amount.
- QR Code Generation: Automatically generates a UPI payment QR code linked to Aditya's PhonePe or other UPI ID, based on the entered amount.
- Secure Transaction Handling: Ensures secure transactions by dynamically generating the QR code, minimizing errors.
- UPI ID Integration: Integrates with Aditya's existing UPI ID (PhonePe or other UPI) to receive payments directly.

## Style Guidelines:

- Primary color: Strong violet (#A34EFF) for digital currency and trust.
- Background color: Light lavender (#F3E8FF), desaturated and bright.
- Accent color: Deep indigo (#3F37C9), darker and saturated to highlight key actions.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look
- Use simple, modern icons to represent payment methods and confirmation.
- Clean and straightforward layout with clear sections for amount input and QR code display.
- Subtle animations to indicate successful QR code generation and payment processing.
